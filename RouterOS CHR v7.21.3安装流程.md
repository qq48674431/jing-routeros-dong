
# RouterOS CHR v7.21.3 安装流程

本指南详细介绍如何使用本仓库提供的一键安装脚本，快速自动地将目标 Linux 机器重装为 RouterOS CHR v7.21.3。

## 一、安装前准备

1. **环境要求**：
   - 具有 root 权限的 Linux 系统（通常在 Debian / Ubuntu 下执行最为兼容）。
   - 由于脚本使用 `losetup`（需支持 `-P` 参数）、`lsblk` 及 `dd` 等工具，建议系统组件尽量齐全且内核版本不要过陈旧。
   - 网络需正常连接（脚本会自动获取当前主路由网卡的 IP 以及网关信息）。
2. **下载脚本**：
   确保你的环境里已就绪 `v7.21.3install.sh` 脚本文件。

## 二、配置与执行安装

### 1. 修改系统默认初始密码（可选）

可以编辑 `v7.21.3install.sh`，修改开头部分的 `ROS_PASSWORD` 变量以设定全新的 admin 密码：

```bash
# 设置你的 ROS 系统密码 (建议修改这里)
ROS_PASSWORD="你的自定义密码"
```

*(默认脚本中的密码为 `Admin112233`)*

### 2. 执行一键安装脚本

在终端中进入脚本所在目录，并运行脚本：

```bash
bash v7.21.3install.sh
```

## 三、脚本安装全自动流程解析

运行脚本后，安装流程将全自动进行，您可以从终端屏幕上观察到具体的进度。主要通过如下几步：

1. **环境检测与镜像下载**：
   脚本会自动检测系统引导方式是处于 UEFI 还是 BIOS 模式，从而从 GitHub 对应的 Release 下载标准版 (`chr-7.21.3.img`) 或 Bios legacy 专用包 (`chr-7.21.3-legacy-bios.img`)。
2. **网络备份防失联机制**：
   脚本将自动提取你当前的有效 IP 地址以及网关路由信息，把这些数据缓存起来。
3. **镜像挂载与离线注入配置**：
   - 脚本使用 `losetup -P` 挂开刚才下载取得的镜像，并识别包含 RouterOS 配置所在的 `/rw` 分区。
   - 在该分区创建一份 `autorun.scr`。这个自动运行脚本包含了你在上一步设定的 admin 密码、将主网卡设为 `wan`、注入前面缓存的自动 IP 和网关、关闭相对不安全的 telnet、保留 ssh (22) 和 winbox 服务开启。
4. **全盘覆盖 (DD 写盘)**：
   脚本将尝试通过 `lsblk` 命令探测当前的物理硬盘设备 (例如 `sda` 或 `vda`)。随后通过强力工具 `dd` 将镜像全速直接写进对应硬盘。
   > **警告**：此过程不可逆，并且将完全清除整个硬盘上原本的全部系统与数据源！请提前做好全部备份！
5. **内核强制重启**：
   顺利写入完成 3 秒后，脚本会越权利用 sysrq 终极命令强制使机器进行复位操作重启动。

等机器再次上电引导完毕后，它就已经变成了一台 RouterOS，您可以使用先前设置的密码，直接通过原有 IP 利用 WinBox 或者是 SSH 接入它。
